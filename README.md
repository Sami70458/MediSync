# Getting Started
 ## Installation
 

Clone the Repository

```bash
  git clone https://github.com/Sami70458/MediSync.git

```
Navigate to the Project Directory
```bash
  cd MediSync
```
Install Dependencies
```bash
  npm i
```
Run the Development Server
```bash
  npm run dev
```
